# MCGC
MC Group YGO Card Set

“植吧MC群”第一次游戏王比赛后脑洞的产物，`MCQ.txt`为卡片设定，余下的`84130811.lua`开始的是各个卡片的代码

基于YgoCore/YgoPro开发

PS：API真难用

CardSet: `0x2222`

## 卡库使用方法
将`mcq.cdb`复制到ygopro目录下的expansions文件夹

## 卡片添加方法
1. 用卡库编辑器（不要问我在哪里下）打开`mcq.cdb`，将卡片信息录入
2. 将对应卡片的脚本复制到游戏目录下的`script`目录
